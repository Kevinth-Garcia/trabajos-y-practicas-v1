Mi nueva [API/REST] con Node.js

Un sistema basico de autenticacion con JWT para el encriptado de contraseñas y proteccion del usuario usando Node.js con las siguientes independencias instaladas:

1. express
2. jsonwebtoken
3. bcrypt
4. dotenv

En este podras observar la estructura para llevar este sistema acabo junto con un ejemplo de archivo .env que por obvias razones no puedo mostrar pero si darte un ejemplo aca.

Para poder usarlo lo unico que necesitas es instalar [Node.js], junto con la extension de ThunderCliente que se encuentra en la pestaña de extensiones de VScode, ejecutar el comando en bash [node(tu_archivo.js)]
y dirigirse al ThunderClient para empezar a realizar las pruebas por POST con el registro, login , y GET para la autenticacion por usuario.

[TECNOLOGIAS]

1. Node.js
2. ThunderClient.
